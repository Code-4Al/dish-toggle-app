# Dish Toggle App

This application is built using **Next.js**, **Server Actions**, and **Prisma**.

## Getting Started

Follow these steps to get a local copy of the project up and running:
 1. Clone the repository
git clone https://github.com/Code-4Al/dish-toggle-app.git

2. Navigate to the project directory
cd dish-toggle-app

3. Install dependencies
npm install

4. Setup Prisma
npx prisma generate
npx prisma migrate dev

After setup, you can start the development server:
npm run dev
